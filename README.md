# 🎲 Math Quest

## Gamified Math Learning Platform

---

#  Overview

Math Quest is an interactive web-based learning platform that transforms mathematics into an engaging adventure. Instead of traditional practice sessions, users progress through topic-based challenges, earn experience points (XP), unlock advanced levels, and collect achievement badges as they master Algebra, Geometry, Trigonometry, and Calculus.

The platform combines gamification with personalized progress tracking to make learning mathematics fun, motivating, and accessible for learners of all levels.

---

#  Features

###  User Authentication
- Secure user registration and login
- Personalized learning experience
- Individual progress tracking

###  Topic-Based Challenges
- Algebra
- Geometry
- Trigonometry
- Calculus
- Progressive learning paths

###  XP & Level System
- Earn experience points for solving questions
- Level up by completing challenges
- Unlock advanced topics as you progress

###  Achievement Badges
- Unlock badges for completing milestones
- Reward-based learning experience
- Encourages consistent practice

###  Progress Tracking
- Saves user progress using LocalStorage
- Resume learning anytime
- Maintains XP, levels, and achievements

###  Responsive User Interface
- Mobile-friendly design
- Clean and intuitive interface
- Optimized for desktop and mobile devices

###  Fast Development Environment
- Powered by Vite for fast loading and development
- Smooth and responsive user experience

# 🚀 Installation

## Prerequisites

- Node.js (v18 or later)
- Git

### Clone the Repository

```bash
git clone https://github.com/sharadhiadiga/mathquest.git

cd mathquest
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
